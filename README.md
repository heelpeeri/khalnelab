# خل نلعب

نسخة MVP نظيفة لمنصة ألعاب عائلية عربية.

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح:
`http://localhost:3000`

## الملاحظات
- الشعار موجود داخل `public/logo.png`
- الواجهة جاهزة ومتصلة بين:
  - الرئيسية
  - الاسم المستعار
  - الغرفة
  - اللعبة
  - النهاية
- هذه نسخة MVP بدون Realtime حقيقي
- مناسبة كنقطة بداية نظيفة بعد المشاكل السابقة
